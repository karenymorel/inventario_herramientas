# 🛠️ Manager de Herramientas Industriales (.NET 8 + PostgreSQL)

Este proyecto demuestra una arquitectura robusta para el control de activos con trazabilidad total.

## 🚀 Características Principales
- **Autenticación OAuth 2.0**: Login seguro integrado con Google Cloud Console.
- **Auditoría Completa**: Registro automático de qué usuario creó o modificó cada activo y en qué fecha exacta.
- **Soft Delete**: Implementación de borrado lógico para preservar la integridad de los datos históricos.
- **Dashboard de Métricas**: Visualización en tiempo real del estado del inventario y alertas de stock crítico.
- **Buscador Dinámico**: Filtrado instantáneo por múltiples criterios (Nombre, Categoría, Estado).
- **Diseño Responsive**: Interfaz moderna en modo oscuro (Dark Mode) optimizada para dispositivos móviles y escritorio.

## 🛠️ Tecnologías Utilizadas
- **Backend**: C# con .NET 8 y ASP.NET Core MVC.
- **Base de Datos**: PostgreSQL.
- **Acceso a Datos**: Dapper (Micro-ORM) para consultas SQL optimizadas.
- **Arquitectura**: Repository Pattern para una separación clara de responsabilidades.
- **Frontend**: Razor Pages, CSS3 Custom Properties (Variables) y JavaScript (jQuery).

## 🔒 Nota modo demo
Este proyecto se encuentra en **Modo Demo**. Las acciones de creación y edición están habilitadas visualmente pero bloqueadas para mantener la integridad de los datos.
